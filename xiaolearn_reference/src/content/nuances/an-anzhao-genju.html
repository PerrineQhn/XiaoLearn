Idée générale

Ces trois formes introduisent un complément du type « selon / d'après / en suivant ». La nuance dépend du type de référence : procédure/règle vs base/preuve.

按 (àn) : selon / en suivant

  • Très courant avec 按时, 按规定, 按顺序.
  • Souvent: suivre une méthode, un plan, un ordre.
请按顺序排队。
Qǐng àn shùnxù páiduì.
Fais la queue dans l'ordre, s'il te plaît.
我们按计划出发。
Wǒmen àn jìhuà chūfā.
Nous partons selon le plan.

按照 (ànzhào) : conformément à

  • Proche de 按, plus explicite et parfois plus formel.
  • Fréquent dans les consignes et règles écrites.
请按照说明书操作。
Qǐng ànzhào shuōmíngshū cāozuò.
Suis le mode d'emploi pour effectuer l'opération.

根据 (gēnjù) : d'après / sur la base de

  • Accent sur la base: expérience, observation, documents, données.
  • Fréquent avec 根据经验, 根据数据, 根据你的描述.
根据数据，我们需要调整方案。
Gēnjù shùjù, wǒmen xūyào tiáozhěng fāng'àn.
D'après les données, nous devons ajuster le plan.

Choix rapide

  • Procédure / ordre / plan -> 按 (ou 按照 pour insister).
  • Base / preuve / données / expérience -> 根据.
