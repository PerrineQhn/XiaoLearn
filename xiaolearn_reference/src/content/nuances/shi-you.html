Vue d'ensemble

Mot Pinyin Sens Négation
是 shì être (identité) 不是
有 yǒu avoir / il y a 没有

是 (shì) - Être / Égalité

是 établit une relation d'identité ou d'égalité entre le sujet et ce qui suit.

Utilisations

  1. Identifier quelqu'un ou quelque chose

他是老师。 Tā shì lǎoshī. Il est professeur.

这是我的书。 Zhè shì wǒ de shū. C'est mon livre.

  1. Confirmer ou affirmer

对，是的。 Duì, shì de. Oui, c'est ça.

Négation de 是

Important : On utilise toujours 不 avec 是, jamais 没 !

他不是学生。 Tā bú shì xuéshēng. Il n'est pas étudiant.

有 (yǒu) - Avoir / Il y a

有 exprime la possession ou l'existence.

Utilisations

  1. Possession

我有一本书。 Wǒ yǒu yì běn shū. J'ai un livre.

她有两个孩子。 Tā yǒu liǎng gè háizi. Elle a deux enfants.

  1. Existence (il y a)

桌子上有一杯咖啡。 Zhuōzi shàng yǒu yì bēi kāfēi. Il y a une tasse de café sur la table.

教室里有很多学生。 Jiàoshì lǐ yǒu hěn duō xuéshēng. Il y a beaucoup d'étudiants dans la salle de classe.

Négation de 有

Important : On utilise toujours 没 avec 有, jamais 不 !

我没有钱。 Wǒ méiyǒu qián. Je n'ai pas d'argent.

这里没有人。 Zhèlǐ méiyǒu rén. Il n'y a personne ici.

Comparaison directe

Français Avec 是 Avec 有
C'est un livre 这是书 ✗
J'ai un livre ✗ 我有书
Il est professeur 他是老师 ✗
Il a un professeur ✗ 他有老师

Erreurs courantes

Erreur 1 : Utiliser 是 pour décrire un lieu

Incorrect Correct
桌子上是一本书 桌子上有一本书

Pour dire "il y a X quelque part", utilisez 有.

Erreur 2 : Utiliser 有 pour identifier

Incorrect Correct
这有我的手机 这是我的手机

Pour dire "c'est X", utilisez 是.

Erreur 3 : Mauvaise négation

Incorrect Correct
没是 不是
不有 没有

Astuce mnémotechnique

  • 是 = "c'EST" (identité, égalité)
  • 有 = "Y a" (possession, existence)

Pensez à : "C'est = 是" et "Il y a = 有"

Exercices

Complétez avec 是 ou 有 :

  1. 我 ___ 中国人。(Je suis chinois)
  2. 你 ___ 手机吗？(Tu as un téléphone ?)
  3. 这 ___ 什么？(C'est quoi ?)
  4. 房间里 ___ 三个人。(Il y a 3 personnes dans la chambre)
Voir les réponses 1. 是 (identité) 2. 有 (possession) 3. 是 (identification) 4. 有 (existence)
