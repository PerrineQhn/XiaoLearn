## Introduction

Cette leçon de la série Chinese Grammar Gems explore comment poser des questions sur la méthode, l'évaluation et le type ou les qualités en chinois. Nous nous concentrerons sur trois mots clés : 怎么 (zěnme), 怎么样 (zěnmeyàng) et 什么样 (shénmeyàng). Ces mots sont utilisés de manière distincte pour exprimer des questions différentes, mais souvent confondues par les apprenants.

## Structure

```
([Sujet]) + 怎么 + [Verb] + ([Objet]) + ？
```

### Exemples

**汉字例句。**
*这个字怎么写？* (Zhège zì zěnme xiě?)
*pīnyīn lì jù.* réciter la pinyin avec les tons diacritiques
这个字怎么写？ (zhège zì zěnme xiě?) 
Traduction française : Comment écris-tu ce caractère ?

**汉字例句。**
*这道菜怎么做？* (Zhè dào cài zěnme zuò?)
*pīnyīn lì jù.* réciter la pinyin avec les tons diacritiques
这道菜怎么做？ (zhè dào cài zěnme zuò?) 
Traduction française : Comment prépare-t-on ce plat ?

## Nuances d'usage

### 怎么 (zěnme) pour la méthode

怎么 (zěnme) est utilisé pour demander comment faire quelque chose. Il se place généralement avant le verbe.

- **Exemple simple** :
  - *这个字怎么写？* (Zhège zì zěnme xiě?)
  - Traduction française : Comment écris-tu ce caractère ?

### 怎么样 (zěnmeyàng) pour l'évaluation

怎么样 (zěnmeyàng) est utilisé pour demander comment quelque chose se passe ou s'est passé. Il se place généralement à la fin de la phrase.

- **Exemple simple** :
  - *这道菜怎么样？* (Zhè dào cài zěnmeyàng?)
  - Traduction française : Comment est ce plat ?

### 什么样 (shénmeyàng) pour le type ou les qualités

什么样 (shénmeyàng) est utilisé pour demander quel type de chose il s'agit ou quelles sont ses qualités. Il se place généralement à la fin de la phrase.

- **Exemple simple** :
  - *这种水果什么样？* (Zhè zhǒng shuǐguǒ shénmeyàng?)
  - Traduction française : Quel type de fruit est-ce ?

## Erreurs courantes

| Incorrect | Correct | Raison |
|-----------|---------|--------|
| 怎么样 (zěnmeyàng) pour demander la méthode | 怎么 (zěnme) | 怎么样用于评价或询问状态，而不是方法。 |
| 什么样 (shénmeyàng) pour demander l'évaluation | 什么样 (shénmeyàng) | 什么样专门用来询问类型或品质。 |

## Points clés à retenir

1. 怎么 (zěnme) est utilisé pour demander la méthode.
2. 怎么样 (zěnmeyàng) est utilisé pour demander l'évaluation ou comment quelque chose se passe.
3. 什么样 (shénmeyàng) est utilisé pour demander le type ou les qualités.

## Exercices

1. **Exercice 1** : Complétez la phrase avec 怎么, 怎么样, ou 什么样。
   - *你为什么没来？* (Nǐ wèishénme méi lái?) 
     - **Réponse correcte** : 你怎么没来？(Nǐ zěnme méi lái?)
   
2. **Exercice 2** : Traduisez les phrases en français.
   - *这个字怎么写？* (Zhège zì zěnme xiě?) 
     - **Réponse correcte** : Comment écris-tu ce caractère ?
   - *这道菜怎么样？* (Zhè dào cài zěnmeyàng?)
     - **Réponse correcte** : Comment est ce plat ?

3. **Exercice 3** : Choisissez la réponse appropriée.
   - Quelle phrase utilise 怎么样？
     - *A. 这本书怎么样？* (Zhè běn shū zěnmeyàng?)
     - *B. 这个字怎么写？* (Zhège zì zěnme xiě?)
     - **Réponse correcte** : A.
   
   - Quelle phrase utilise 什么样？
     - *A. 这本书怎么样？* (Zhè běn shū zěnmeyàng?)
     - *B. 这种水果什么样？* (Zhè zhǒng shuǐguǒ shénmeyàng?)
     - **Réponse correcte** : B.
