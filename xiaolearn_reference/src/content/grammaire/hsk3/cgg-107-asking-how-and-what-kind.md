## Introduction

Dans ce texte, nous allons explorer comment poser des questions sur la méthode, l'évaluation et le type d'une chose en chinois. Nous verrons les nuances de ces expressions et comment éviter certaines erreurs courantes.

## Structure

```
怎么 (zěnme) - 方法
怎么样 (zěnmeyàng) - 评价
什么样 (shénmeyàng) - 类型或品质
```

### Exemples

**汉字例句。**
*怎么写？*（zěnme xiě?） *—— 这个字怎么写？*（zhège zì zěnme xiě?）
*pīnyīn lì jù.*
Comment écrire ce caractère ?
Traduction française.

**汉字例句。**
*怎么做？*（zěnme zuò?） *—— 这道菜怎么做？*（zhè dào cài zěnme zuò?）
*pīnyīn lì jù.*
Comment faire cette recette ?
Traduction française.

**汉字例句。**
*什么样？*（shénmeyàng?） *—— 你穿什么衣服？*（nǐ chuān shénmeyàng yīfu?）
*pīnyīn lì jù.*
Quelle sorte d'habits portes-tu ?
Traduction française.

## Nuances d'usage

### 怎么 (zěnme) - 方法
怎么用于询问做某事的方法，通常在口语中使用。当问指令或帮助时，主语常常省略。

**汉字例句。**
*怎么写？*（zěnme xiě?）
*pīnyīn lì jù.*
Comment écrire ce caractère ?
Traduction française.

### 怎么样 (zěnmeyàng) - 评价
怎么样用于询问对某事的评价，通常在句子末尾。

**汉字例句。**
*你去哪儿？怎么走？*（nǐ qù nǎr? zěnme zǒu?）
*pīnyīn lì jù.*
Où vas-tu ? Comment y aller ?
Traduction française.

### 什么样 (shénmeyàng) - 类型或品质
什么样用于询问某物的类型或品质。

**汉字例句。**
*你穿什么衣服？*（nǐ chuān shénmeyàng yīfu?）
*pīnyīn lì jù.*
Quelle sorte d'habits portes-tu ?
Traduction française.

## Erreurs courantes

| Incorrect | Correct | Raison |
|-----------|---------|--------|
| *怎么*（zěnme） + *写？*（xiě?） *—— 这个字怎么样写？*（zhège zì zěnmeyàng xiě?） | *怎么*（zěnme） + *写？*（xiě?） | 怎么用于询问方法，而不是评价 |
| *什么样*（shénmeyàng） + *走？*（zǒu?） *—— 你去哪儿怎么样走？*（nǐ qù nǎr zěnmeyàng zǒu?） | *怎么*（zěnme） + *走？*（zǒu?） | 怎么样用于询问评价，而不是方法 |

## Points clés à retenir

1. 怎么 (zěnme) est utilisé pour demander une méthode.
2. 怎么样 (zěnmeyàng) est utilisé pour évaluer une situation.
3. 什么样 (shénmeyàng) est utilisé pour demander le type ou les qualités d'une chose.
