## Introduction

En chinois, il existe principalement deux façons de former une négation : avec **不** (bù) et **没** (méi). Choisir le bon dépend du contexte et du type d'action.

## 不 (bù) - Négation générale

**不** est utilisé pour :

- Les états permanents ou habituels
- Les actions présentes ou futures
- Les adjectifs
- Le verbe 是 (être)

### Structure

```
Sujet + 不 + Verbe/Adjectif
```

### Exemples

**我不是中国人。**
*Wǒ bú shì Zhōngguó rén.*
Je ne suis pas chinois.

**他不喜欢咖啡。**
*Tā bù xǐhuān kāfēi.*
Il n'aime pas le café.

**今天不冷。**
*Jīntiān bù lěng.*
Aujourd'hui il ne fait pas froid.

**我明天不去。**
*Wǒ míngtiān bú qù.*
Je n'irai pas demain.

> **Note sur la prononciation :** 不 se prononce normalement "bù" (4e ton), mais devient "bú" (2e ton) devant un autre 4e ton.

## 没 (méi) - Négation du passé

**没** (ou sa forme complète **没有**) est utilisé pour :

- Les actions qui ne se sont pas produites dans le passé
- La possession (没有 = ne pas avoir)

### Structure

```
Sujet + 没(有) + Verbe
```

### Exemples - Actions passées

**我昨天没去学校。**
*Wǒ zuótiān méi qù xuéxiào.*
Je ne suis pas allé à l'école hier.

**他没吃早饭。**
*Tā méi chī zǎofàn.*
Il n'a pas mangé le petit-déjeuner.

**她没看那本书。**
*Tā méi kàn nà běn shū.*
Elle n'a pas lu ce livre.

### Exemples - Possession

**我没有钱。**
*Wǒ méiyǒu qián.*
Je n'ai pas d'argent.

**他没有车。**
*Tā méiyǒu chē.*
Il n'a pas de voiture.

## Tableau comparatif

| Contexte | Mot | Exemple |
|----------|-----|---------|
| État/habitude | 不 | 我**不**喜欢 (je n'aime pas) |
| Action future | 不 | 我明天**不**去 (je n'irai pas) |
| Avec 是 | 不 | 他**不**是老师 (il n'est pas prof) |
| Action passée | 没 | 我昨天**没**去 (je n'y suis pas allé) |
| Possession | 没有 | 我**没有**时间 (je n'ai pas le temps) |

## Erreurs courantes

| Incorrect | Correct | Raison |
|-----------|---------|--------|
| 我昨天不去 | 我昨天**没**去 | Action passée → 没 |
| 我没是学生 | 我**不**是学生 | 是 → toujours 不 |
| 我不有钱 | 我**没有**钱 | Possession → 没有 |

## Points clés à retenir

1. **不** pour : états, habitudes, futur, 是, adjectifs
2. **没** pour : actions passées, possession (没有)
3. On ne dit JAMAIS "不有" - c'est toujours "没有"
4. On ne dit JAMAIS "没是" - c'est toujours "不是"

## Exercices

Choisissez 不 ou 没 :

1. 我 ___ 是法国人。(Je ne suis pas français)
2. 他昨天 ___ 来。(Il n'est pas venu hier)
3. 她 ___ 喜欢猫。(Elle n'aime pas les chats)
4. 我们 ___ 有时间。(Nous n'avons pas le temps)

1. 不 (avec 是)
2. 没 (action passée)
3. 不 (état/goût)
4. 没 (possession)
