## Introduction

La structure de phrase chinoise de base suit un ordre **Sujet-Verbe-Objet (SVO)**, ce qui est une bonne nouvelle pour les francophones car c'est similaire au français !

## Structure de base

```
Sujet + Verbe + Objet
```

### Exemples simples

**我吃饭。**
*Wǒ chī fàn.*
Je mange (du riz).

**他喝茶。**
*Tā hē chá.*
Il boit du thé.

**她看书。**
*Tā kàn shū.*
Elle lit un livre.

## Avec un complément de temps

En chinois, les expressions de temps se placent **avant le verbe**, souvent juste après le sujet.

```
Sujet + Temps + Verbe + Objet
```

### Exemples

**我今天吃面条。**
*Wǒ jīntiān chī miàntiáo.*
Je mange des nouilles aujourd'hui.

**他明天去北京。**
*Tā míngtiān qù Běijīng.*
Il va à Pékin demain.

**她每天学习中文。**
*Tā měitiān xuéxí Zhōngwén.*
Elle étudie le chinois tous les jours.

## Avec un adverbe

Les adverbes comme 也 (yě - aussi), 都 (dōu - tous), 很 (hěn - très) se placent **avant le verbe**.

```
Sujet + Adverbe + Verbe + Objet
```

### Exemples

**我也喜欢咖啡。**
*Wǒ yě xǐhuān kāfēi.*
J'aime aussi le café.

**他们都是学生。**
*Tāmen dōu shì xuéshēng.*
Ils sont tous étudiants.

**她很高兴。**
*Tā hěn gāoxìng.*
Elle est très contente.

## Points importants

> **Attention !** Contrairement au français, les adverbes ne se placent JAMAIS après le verbe en chinois.

| Français | Chinois |
|----------|---------|
| Je mange **aussi** | 我**也**吃 (wǒ **yě** chī) |
| Il va **souvent** | 他**常常**去 (tā **chángcháng** qù) |

## Résumé

1. L'ordre de base est **SVO** : Sujet + Verbe + Objet
2. Le temps se place **entre le sujet et le verbe**
3. Les adverbes se placent **avant le verbe**
4. Cette structure est la fondation de toutes les phrases chinoises
