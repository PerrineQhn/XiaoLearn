## Introduction

La particule **的** (de) est l'un des mots les plus fréquents en chinois mandarin. Elle joue un rôle similaire à la préposition "de" en français pour exprimer la possession, mais elle a aussi d'autres usages importants.

## 1. Exprimer la possession

La structure de base est :

```
Possesseur + 的 + Objet possédé
```

### Exemples

**我的书**
*wǒ de shū*
mon livre

**他的妈妈**
*tā de māma*
sa mère

**老师的名字**
*lǎoshī de míngzi*
le nom du professeur

**中国的首都**
*Zhōngguó de shǒudū*
la capitale de la Chine

## 2. Avec les adjectifs

Quand un adjectif modifie un nom, on utilise généralement 的 entre eux :

```
Adjectif + 的 + Nom
```

### Exemples

**漂亮的花**
*piàoliang de huā*
une belle fleur

**好吃的菜**
*hǎochī de cài*
un plat délicieux

**有名的人**
*yǒumíng de rén*
une personne célèbre

## 3. Omission de 的

Dans certains cas, 的 peut être omis :

### Avec les relations familiales proches

**我妈妈** (au lieu de 我的妈妈)
*wǒ māma*
ma mère

**他爸爸**
*tā bàba*
son père

### Avec les adjectifs d'une syllabe

**大房子** (au lieu de 大的房子)
*dà fángzi*
grande maison

**好人**
*hǎo rén*
bonne personne

> **Astuce :** Pour les adjectifs de deux syllabes ou plus, 的 est généralement obligatoire.

## 4. Comparaison avec le français

| Français | Chinois |
|----------|---------|
| le livre **de** Marie | 玛丽**的**书 |
| **mon** ami | 我**的**朋友 |
| un **beau** jardin | 漂亮**的**花园 |

## Points clés à retenir

1. 的 se place toujours **entre** le modificateur et le nom
2. L'ordre est l'inverse du français pour la possession : possesseur vient **avant**
3. 的 peut être omis avec les relations proches et les adjectifs courts
4. Quand vous hésitez, utilisez 的 - c'est rarement faux !

## Exercices pratiques

Essayez de traduire :

1. Ma maison → ?
2. Le chat noir → ?
3. L'ami de mon frère → ?

<details>
<summary>Voir les réponses</summary>

1. 我的房子 (wǒ de fángzi)
2. 黑的猫 ou 黑猫 (hēi de māo ou hēi māo)
3. 我哥哥的朋友 (wǒ gēge de péngyou)

</details>
