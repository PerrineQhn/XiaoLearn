祈使句

HSK1 句子成分 / 句类 细目 : 祈使句

Idée générale

La phrase impérative exprime un ordre, une demande ou une suggestion. 请 adoucit l'impératif. 不要 exprime une interdiction.

Structure

(请) + Verbe
不要 / 别 + Verbe (interdiction)

Comment l'utiliser

  • 请 rend l'impératif poli : 请坐 (Asseyez-vous, s'il vous plaît).
  • 不要 pour les interdictions : 不要说话！

Exemples

请坐！
Asseyez-vous, s'il vous plaît !
不要说话！
Ne parle pas !
快来！
Viens vite !

Notes

  • Sans 请, l'impératif peut être brusque.

Pièges fréquents

  • Ne pas oublier 请 pour être poli dans les demandes.
