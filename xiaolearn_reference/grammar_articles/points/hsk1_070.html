时间表示法 — （2）钟点表示法

HSK1 特殊表达法 / 时间表示法

Idée générale

L'heure s'exprime avec 点 (heure), 分 (minute), 半 (demie), 刻 (quart). L'heure se place AVANT le verbe.

Structure

Nombre + 点 (+ Nombre + 分) | Nombre + 点 + 半

Comment l'utiliser

  • 三点 (3h), 三点半 (3h30), 三点一刻 (3h15).

Exemples

现在几点？
Quelle heure est-il ?
现在三点半。
Il est trois heures et demie.
我八点上课。
J'ai cours à huit heures.

Notes

  • L'heure (moment) se place AVANT le verbe : 我八点去.

Pièges fréquents

  • *我去八点 → 我八点去 : le moment temporel précède le verbe.
