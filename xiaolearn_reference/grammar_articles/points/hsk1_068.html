数的表达法 — 序数表达法1：第+数词

HSK1 特殊表达法 / 数的表达法

Idée générale

Les ordinaux se forment avec 第 + nombre : 第一 (premier), 第二 (deuxième), etc.

Structure

第 + Nombre (+ Classificateur + Nom)

Comment l'utiliser

  • Pour les classements et rangs.

Exemples

他是第一名。
Il est premier.
这是我第一次来中国。
C'est ma première fois en Chine.
请打开第三页。
Ouvrez la page trois, s'il vous plaît.

Notes

  • 第 est obligatoire pour former un ordinal.

Pièges fréquents

  • 一次 (une fois) ≠ 第一次 (la première fois) : 第 change le sens.
