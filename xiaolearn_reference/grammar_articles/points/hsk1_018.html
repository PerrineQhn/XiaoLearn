频率副词 — 再1

HSK1 词类 / 副词 细目 : 频率副词

Idée générale

L'adverbe 再 indique la répétition future d'une action (« encore une fois, à nouveau » dans le futur). Il se place avant le verbe.

Structure

Sujet + 再 + Verbe

Comment l'utiliser

  • 再 pour une action qui va se répéter dans le futur.
  • 再见 = au revoir (littéralement : « on se reverra »).

Exemples

我明天再来。
Je reviendrai demain.
请再说一次。
Veuillez le redire une fois.
再见！
Au revoir !
你再想想。
Réfléchis-y encore.

Notes

  • 再 est pour le FUTUR ; pour la répétition passée, on utilise 又 (HSK supérieur).

Pièges fréquents

  • Ne pas utiliser 再 pour le passé : *我昨天再去了 → 我昨天又去了.
