数词 — 一、二、两、三…

HSK1 词类 / 数词

Idée générale

Les nombres de base : 一 à 十, 百 (cent), 千 (mille). 二 est le chiffre utilisé dans les nombres composés, 两 s'emploie devant les classificateurs. 半 = demi, 零 = zéro.

Structure

Nombre + Classificateur + Nom
两 + Classificateur (pas 二 + Classificateur)

Comment l'utiliser

  • 两 remplace 二 devant les classificateurs : 两个人 (deux personnes), pas *二个人.
  • 二 dans les nombres composés : 二十 (vingt), 一百二十 (cent vingt).
  • 半 = demi : 半个小时 (une demi-heure) ; 零 pour les trous : 一百零五 (105).

Exemples

我有两个姐姐。
J'ai deux grandes sœurs.
一百零三个人。
Cent trois personnes.
半个小时。
Une demi-heure.
三十五块钱。
Trente-cinq yuans.

Notes

  • 两 remplace 二 devant les classificateurs ; pour 200, on dit 两百.
  • 零 est nécessaire pour les trous : 一百零五 (105), pas *一百五 (qui signifie 150).

Pièges fréquents

  • *二个 → 两个 : devant un classificateur, c'est toujours 两.
  • 一百五 = 150 (raccourci oral) ; 一百零五 = 105. Ne pas confondre.
