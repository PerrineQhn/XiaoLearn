指示代词 — 这、那、这儿、那儿…

HSK1 词类 / 代词 细目 : 指示代词

Idée générale

Les démonstratifs : 这 (ce/ceci - proche), 那 (ce/cela - éloigné). 这儿/这里 (ici), 那儿/那里 (là-bas). 这些/那些 (ceux-ci/ceux-là). 有的/有些 (certains).

Structure

这 / 那 + Classificateur + Nom
这些 / 那些 + Nom

Comment l'utiliser

  • 这 pour ce qui est proche, 那 pour ce qui est éloigné.
  • 这儿/这里 et 那儿/那里 sont interchangeables (这儿 plus oral).
  • 有的/有些 = certains, dans un ensemble connu.

Exemples

这个苹果很好吃。
Cette pomme est délicieuse.
那本书是我的。
Ce livre-là est le mien.
你家在哪里？
Où habites-tu ?
有的人喜欢看书。
Certaines personnes aiment lire.

Notes

  • 哪 (lequel/où) est un interrogatif, 那 (celui-là) est un démonstratif.
  • 这儿/这里 sont interchangeables ; 这儿 est plus oral, 这里 plus standard.

Pièges fréquents

  • Ne pas oublier le classificateur : *这书 → 这本书.
  • Ne pas confondre 哪 (interrogatif, quel) et 那 (démonstratif, ce...là).
