动词重叠 — AA、A一A、A了A、ABAB

HSK2 词类 / 动词 细目 : 动词重叠

Idée générale

La **réduplication du verbe** (动词重叠) adoucit l'action : « faire un peu / essayer / brièvement » (看一看, 说说).

Éléments concernés

AA A一A A了A ABAB

Exemples

你看一看这个。
Regarde un peu ça.

Pièges fréquents

  • Ne pas confondre la fonction (catégorie / structure) avec le vocabulaire : les mots listés suivent un même comportement, mais le sens précis peut varier.
  • Vérifier la place dans la phrase (avant le verbe, après le verbe, dans le groupe nominal, etc.).
  • Quand un schéma implique un complément (résultat, direction, durée…), éviter une phrase “trop courte” qui sonne artificielle.

Notes utiles

  • 了 — Aspect / changement d'état : Marque un événement accompli ou un changement de situation. Attention : 了 n'est pas obligatoire dans toutes les phrases au passé.
