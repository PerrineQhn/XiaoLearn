谓语 — 动词或动词性短语、形容词或形容词性短语作 谓语

HSK1 句子成分 / 谓语

Idée générale

Le prédicat verbal utilise un verbe ou un groupe verbal. Le prédicat adjectival utilise un adjectif directement, SANS le verbe « être » (是). 很 sert souvent de lien neutre.

Structure

Sujet + Verbe (+ Objet)
Sujet + (很) + Adjectif

Comment l'utiliser

  • Le verbe sert de prédicat avec son objet : 我学习中文.
  • L'adjectif sert de prédicat avec 很 (neutre) : 她很漂亮.
  • Sans 很, un adjectif seul implique une comparaison.

Exemples

我学习中文。
J'étudie le chinois.
她很漂亮。
Elle est jolie.
天气很冷。
Il fait froid.

Notes

  • Un adjectif seul sans 很 implique une comparaison : 她漂亮 ≈ « elle est plus belle ».

Pièges fréquents

  • *她是漂亮 → 她很漂亮 : pas de 是 devant un adjectif prédicat.
