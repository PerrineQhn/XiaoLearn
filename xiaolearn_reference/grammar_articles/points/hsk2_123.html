数量补语2 — （1）表示动作持续的时间

HSK2 句子成分 / 补语 细目 : 数量补语2

Idée générale

Les **compléments de quantité** (数量补语) indiquent combien / combien de fois / combien de temps : V + 数量 + (名/次/遍/分钟…), ou V + 了 + 时量…

Éléments concernés

（1）表示动作持续的时间

Structure(s)

（1）表示动作持续的时间

Exemples

我今天学习中文。
Aujourd'hui j'étudie le chinois.

Pièges fréquents

  • Ne pas confondre la fonction (catégorie / structure) avec le vocabulaire : les mots listés suivent un même comportement, mais le sens précis peut varier.
  • Vérifier la place dans la phrase (avant le verbe, après le verbe, dans le groupe nominal, etc.).
  • Quand un schéma implique un complément (résultat, direction, durée…), éviter une phrase “trop courte” qui sonne artificielle.

Notes utiles

  • 的 — Détermination : Relie un modifieur à un nom (我的书). Attention à la place des groupes longs.
