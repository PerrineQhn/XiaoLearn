基本结构类型 — （3）动宾短语

HSK1 短语 / 结构类型 细目 : 基本结构类型

Idée générale

Le groupe verbe-objet (动宾短语) associe un verbe à son complément d'objet. C'est la structure de base pour exprimer une action.

Structure

Verbe + Objet

Comment l'utiliser

  • L'objet suit directement le verbe.
  • Structure fondamentale : 吃饭 (manger), 看书 (lire), 喝水 (boire de l'eau).

Exemples

吃饭。
Manger.
看书。
Lire.
说中文。
Parler chinois.
喝水。
Boire de l'eau.

Notes

  • Certains groupes verbe-objet sont des 离合词 qui se séparent (看病, 睡觉).

Pièges fréquents

  • L'objet est toujours APRÈS le verbe : *中文说 → 说中文.
