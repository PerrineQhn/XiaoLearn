陈述句

HSK1 句子成分 / 句类 细目 : 陈述句

Idée générale

La phrase déclarative (陈述句) exprime un fait, une opinion ou un état. C'est le type de phrase le plus courant.

Structure

Sujet + Prédicat (+ Objet)

Comment l'utiliser

  • Pour affirmer ou nier : 我是学生 / 我不是学生.

Exemples

我是法国人。
Je suis français(e).
她不喜欢咖啡。
Elle n'aime pas le café.
今天天气很好。
Le temps est beau aujourd'hui.

Notes

  • La négation se fait avec 不 ou 没(有) selon le contexte.

Pièges fréquents

  • L'ordre des mots est strict : Sujet + Temps + Lieu + Verbe + Objet.
