“是”字句1 — 表示等同或类属

HSK1 句子成分 / 特殊句型 细目 : “是”字句1

Idée générale

La structure 是 exprime l'identité (A est B) ou l'appartenance à une catégorie. 是 relie le sujet à un complément nominal.

Structure

A + 是 + B (noms uniquement)
A + 不是 + B (négation)

Comment l'utiliser

  • Pour identifier : 我是学生 ; pour nier : 我不是老师.

Exemples

我是中国人。
Je suis chinois(e).
这是我的书。
C'est mon livre.
她不是老师。
Elle n'est pas professeur.

Notes

  • 是 s'utilise uniquement avec des noms/groupes nominaux, pas avec des adjectifs.

Pièges fréquents

  • *她是漂亮 → 她很漂亮 : 是 ne s'utilise pas devant un adjectif.
