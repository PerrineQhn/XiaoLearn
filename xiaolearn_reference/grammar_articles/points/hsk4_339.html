数的表示法 — 小数、分数、百分数、倍数的表示法

HSK4 特殊表达法 / 数的表示法

Idée générale

Cette fiche présente 4 façons essentielles d'exprimer des nombres en chinois : décimaux (小数), fractions (分数), pourcentages (百分数) et multiples (倍数). Ce sont des structures très fréquentes à l'oral, dans les médias et dans les textes académiques.

Éléments concernés

小数 分数 百分数 倍数

Structure(s)

小数 (décimal) : X点Y
分数 (fraction) : B分之A
百分数 (pourcentage) : 百分之X
倍数 (multiple) : A是B的X倍

Comment l'utiliser

  • Pour les décimaux, on lit chaque chiffre après 点 : 3.14 → 三点一四.
  • Pour les fractions, l'ordre est dénominateur puis numérateur : 3/4 → 四分之三.
  • Pour les pourcentages, on utilise 百分之 + nombre : 20% → 百分之二十.
  • Pour les multiples, la structure standard est A是B的X倍 (A est X fois B).

Exemples

这个产品的合格率是百分之九十八。
Le taux de conformité de ce produit est de 98 %.
三分之一的员工会说中文。
Un tiers des employés parle chinois.
今年的利润比去年高了百分之十五。
Cette année, le bénéfice est supérieur de 15 % à celui de l'année dernière.
这个房间是那个房间的两倍大。
Cette pièce est deux fois plus grande que l'autre.
今天气温是零下二点五度。
La température est de -2,5 degrés aujourd'hui.

Pièges fréquents

  • 分数 peut signifier "fraction" en maths, mais aussi "note d'examen" selon le contexte.
  • Dans une fraction, ne pas inverser l'ordre : 四分之三 (3/4), pas *三分之四.
  • A是B的两倍 signifie "A = 2 × B". Cette structure exprime déjà une comparaison complète.
  • Ne pas confondre pourcentage (百分之...) et nombre décimal (点...).

Notes utiles

  • Forme proche : A比B大一倍 exprime généralement la même idée que A是B的两倍 (A est deux fois B).
  • À l'oral rapide, certains locuteurs simplifient les pourcentages, mais en contexte formel, privilégier 百分之....
