否定副词 — 不、没（有）、不要

HSK1 词类 / 副词 细目 : 否定副词

Idée générale

不 nie le présent/futur et les états habituels. 没（有）nie le passé ou la possession. 不要 exprime une interdiction.

Structure

不 + Verbe/Adj (présent, futur, habitude)
没(有) + Verbe (passé, possession)
不要 + Verbe (interdiction)

Comment l'utiliser

  • 不 pour les états permanents, habitudes et le futur : 我不喜欢 (je n'aime pas).
  • 没（有）pour les événements passés : 他昨天没来 (il n'est pas venu hier).
  • 不要 pour les ordres négatifs : 不要说话！(Ne parle pas !).

Exemples

我不喜欢吃鱼。
Je n'aime pas manger du poisson.
他昨天没来。
Il n'est pas venu hier.
我没有钱。
Je n'ai pas d'argent.
不要说话！
Ne parle pas !

Notes

  • 不 + 是 = ce n'est pas ; 没 + 有 = ne pas avoir.
  • On ne dit jamais *不了 pour nier le passé : il faut 没.

Pièges fréquents

  • *我不去了昨天 → 我昨天没去 : le passé se nie avec 没, pas avec 不.
  • *我不有钱 → 我没有钱 : 有 se nie toujours avec 没, jamais avec 不.
