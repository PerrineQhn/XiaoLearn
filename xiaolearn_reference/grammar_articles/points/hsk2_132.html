比较句1 — （2）A比B+形容词+数量补语、程度补语

HSK2 句子成分 / 句类 细目 : 比较句1

Idée générale

Les **phrases comparatives** permettent d'exprimer une comparaison (supériorité, égalité, différence, progression…). Les marqueurs typiques sont **比**, **跟…一样**, **没有**, **越来越**, **越…越…**.

Éléments concernés

（2）A比B+形容词+数量补语 程度补语

Structure(s)

（2）A比B+形容词+数量补语

Exemples

今天比昨天冷。
Aujourd'hui il fait plus froid qu'hier.

Pièges fréquents

  • Ne pas confondre la fonction (catégorie / structure) avec le vocabulaire : les mots listés suivent un même comportement, mais le sens précis peut varier.
  • Vérifier la place dans la phrase (avant le verbe, après le verbe, dans le groupe nominal, etc.).
  • Quand un schéma implique un complément (résultat, direction, durée…), éviter une phrase “trop courte” qui sonne artificielle.

Notes utiles

  • 比 — Comparaison : A + 比 + B + adj/VP : plus… que…
