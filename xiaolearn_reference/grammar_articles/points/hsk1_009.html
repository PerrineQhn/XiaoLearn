人称代词 — 我、你、您、他…

HSK1 词类 / 代词 细目 : 人称代词

Idée générale

Les pronoms personnels : 我 (je/moi), 你 (tu/toi), 您 (vous - poli), 他 (il), 她 (elle), 它 (il/elle - non humain). Le pluriel se forme avec 们. 大家 signifie « tout le monde ».

Structure

Singulier : 我, 你, 您, 他, 她, 它
Pluriel : 我们, 你们, 他们, 她们, 它们
Collectif : 大家

Comment l'utiliser

  • 您 est la forme polie de 你, utilisée avec les aînés, supérieurs ou inconnus.
  • 他们 peut désigner un groupe mixte (hommes et femmes).
  • 大家 = tout le monde, s'utilise comme sujet : 大家好！ (Bonjour tout le monde !).

Exemples

我是中国人。
Je suis chinois(e).
您贵姓？
Quel est votre nom de famille ? (poli)
他们都是学生。
Ils sont tous étudiants.
大家好！
Bonjour tout le monde !

Notes

  • 他, 她 et 它 se prononcent tous tā : la différence n'existe qu'à l'écrit.
  • Le sujet est souvent omis quand le contexte est clair.

Pièges fréquents

  • Utiliser 你 avec des supérieurs ou des aînés est impoli : préférer 您.
  • 他们 est le pluriel par défaut (y compris mixte) ; 她们 = uniquement des femmes.
