存现句1 — （1）所处+是+名词

HSK1 句子成分 / 特殊句型 细目 : 存现句1

Idée générale

La phrase d'existence avec 是 identifie ce qui se trouve à un endroit. Le lieu est en tête de phrase.

Structure

Lieu + 是 + Nom

Comment l'utiliser

  • Pour identifier ce qu'il y a à un endroit précis.

Exemples

前面是学校。
Devant, c'est l'école.
左边是商店。
À gauche, c'est le magasin.
对面是银行。
En face, c'est la banque.

Notes

  • 是 pour identifier/caractériser un lieu ; 有 pour indiquer l'existence quantifiée.

Pièges fréquents

  • Distinguer : 前面是学校 (identification) vs 前面有一个学校 (existence).
