离合词 — 看病、睡觉、说话、上课…

HSK1 词类 / 动词 细目 : 离合词

Idée générale

Les verbes séparables (离合词) sont des verbes de deux caractères qui peuvent se séparer pour insérer des éléments entre eux (了, un complément de durée, etc.). Exemples : 看病, 睡觉, 说话, 上课.

Structure

Normal : 看病 (voir le médecin)
Séparé : 看了一次病 (vu le médecin une fois)

Comment l'utiliser

  • On sépare le verbe pour insérer 了, un complément de nombre ou de durée.
  • 看病 → 看了一次病 ; 睡觉 → 睡了八个小时觉 ; 说话 → 说了两句话.
  • On ne peut PAS ajouter un objet direct après un 离合词 : *看病了病人 est incorrect.

Exemples

我昨天看了一次病。
Je suis allé(e) chez le médecin hier.
他每天睡八个小时觉。
Il dort huit heures par jour.
你在跟谁说话？
Avec qui parles-tu ?
我们上了三节课。
Nous avons eu trois cours.

Notes

  • Le 2e caractère du 离合词 est en fait l'objet interne du verbe.
  • Pour ajouter un vrai objet, il faut restructurer : 给病人看病 (examiner un patient).

Pièges fréquents

  • Ne pas traiter les 离合词 comme des verbes ordinaires : *他看病了我 → 他给我看了病.
  • Le complément s'insère ENTRE les deux caractères : *睡觉了八小时 → 睡了八个小时觉.
