“有”字句1 — 表示领有

HSK1 句子成分 / 特殊句型 细目 : “有”字句1

Idée générale

La structure 有 exprime la possession (« avoir »). La négation est toujours 没有, jamais *不有.

Structure

Sujet + 有 + Objet
Sujet + 没有 + Objet (négation)

Comment l'utiliser

  • Pour la possession : 我有两本书.
  • Négation toujours avec 没 : 我没有钱.

Exemples

我有一个弟弟。
J'ai un petit frère.
你有没有时间？
As-tu du temps ?
她没有车。
Elle n'a pas de voiture.

Notes

  • La négation de 有 est TOUJOURS 没有, jamais *不有.

Pièges fréquents

  • *我不有钱 → 我没有钱 : 有 se nie toujours avec 没.
