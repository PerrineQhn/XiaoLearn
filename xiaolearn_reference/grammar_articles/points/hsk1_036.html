谓语 — 名词、代词、数词或数量短语作谓语

HSK1 句子成分 / 谓语

Idée générale

En chinois, un nom, un pronom, un nombre ou un groupe numéral peut servir de prédicat SANS verbe « être ». C'est la phrase nominale, utilisée pour les dates, l'âge, les prix.

Structure

Sujet + Nom / Nombre (sans 是)

Comment l'utiliser

  • Pour les dates : 今天星期三 (Aujourd'hui mercredi).
  • Pour l'âge : 她二十岁 (Elle a 20 ans).
  • Pour les prix : 这个三十块 (Ça coûte 30 yuans).

Exemples

今天星期三。
Aujourd'hui c'est mercredi.
她二十岁。
Elle a vingt ans.
这本书三十块钱。
Ce livre coûte trente yuans.
现在八点。
Il est huit heures.

Notes

  • Pas besoin de 是 pour les dates, heures, âges et prix.

Pièges fréquents

  • 今天是星期三 est acceptable mais moins naturel que 今天星期三.
