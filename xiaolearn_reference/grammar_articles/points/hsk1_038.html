宾语 — 名词、代词或名词性短语作宾语

HSK1 句子成分 / 宾语

Idée générale

L'objet (宾语) se place après le verbe. Il est généralement un nom, un pronom ou un groupe nominal, et désigne la cible de l'action.

Structure

Sujet + Verbe + Objet (nom / pronom / groupe nominal)

Comment l'utiliser

  • L'objet suit directement le verbe ; pas de préposition pour l'objet direct.

Exemples

我吃苹果。
Je mange une pomme.
他看电影。
Il regarde un film.
我们学习中文。
Nous étudions le chinois.

Notes

  • Certains verbes prennent deux objets : 老师教我们中文.

Pièges fréquents

  • L'objet est toujours APRÈS le verbe en chinois.
