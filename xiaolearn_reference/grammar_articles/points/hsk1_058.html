双宾语句1 — 主语+动词+宾语1+宾语2

HSK1 句子成分 / 特殊句型 细目 : 双宾语句1

Idée générale

La phrase à double objet : un verbe suivi de deux compléments (objet indirect = personne, objet direct = chose). Verbes typiques : 给, 教, 告诉.

Structure

Sujet + Verbe + Obj. Indirect (personne) + Obj. Direct (chose)

Comment l'utiliser

  • 给 + personne + chose ; 教 + personne + matière ; 告诉 + personne + info.

Exemples

他给我一本书。
Il me donne un livre.
老师教我们中文。
Le professeur nous enseigne le chinois.
请告诉我你的名字。
Dis-moi ton nom, s'il te plaît.

Notes

  • L'objet indirect (personne) vient AVANT l'objet direct (chose).

Pièges fréquents

  • *他给一本书我 → 他给我一本书 : la personne précède la chose.
