单句 — 非主谓句

HSK1 句子成分 / 句型 细目 : 单句

Idée générale

Les phrases sans sujet explicite : exclamations, interjections, salutations, phrases impersonnelles (météo, etc.).

Structure

Interjection / Exclamation / Phrase elliptique

Comment l'utiliser

  • Salutations, réactions, météo impersonnelle.

Exemples

你好！
Bonjour !
谢谢！
Merci !
下雨了。
Il pleut.
对不起。
Pardon.

Notes

  • Très courantes dans la communication quotidienne.

Pièges fréquents

  • Ne pas forcer un sujet quand le chinois n'en a pas besoin : 下雨了 suffit.
