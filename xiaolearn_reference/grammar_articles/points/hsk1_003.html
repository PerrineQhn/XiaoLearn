方位名词 — 上、下、里、外…

HSK1 词类 / 名词 细目 : 方位名词

Idée générale

Les noms de position 上 (sur), 下 (sous), 里 (dedans), 外 (dehors), 前 (devant) et 后 (derrière) se placent APRÈS un nom pour indiquer un emplacement.

Structure

Nom + 上 / 下 / 里 / 外 / 前 / 后

Comment l'utiliser

  • Toujours placés après le nom de référence : 桌子上 (sur la table), 学校里 (dans l'école).
  • Souvent précédés de 在 pour localiser une action : 在学校里学习 (étudier à l'école).
  • Ordre inverse du français : « sur la table » = 桌子上 (table + sur).

Exemples

书在桌子上。
Le livre est sur la table.
猫在床下。
Le chat est sous le lit.
他在房间里。
Il est dans la chambre.
学校前有一棵树。
Il y a un arbre devant l'école.

Notes

  • Contrairement au français, le mot de position vient APRÈS le nom de référence.
  • 里 est indispensable pour « dans » : 房间里 (dans la chambre), pas juste 房间.

Pièges fréquents

  • Ne pas dire *上桌子, mais 桌子上 : le nom précède toujours le mot de position.
  • Ne pas oublier 里 pour exprimer « dans » : *在学校 (ambigu) → 在学校里 (dans l'école).

Éléments concernés

上 下 里 外 前 后
