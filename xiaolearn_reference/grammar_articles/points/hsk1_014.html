时量词 — 日、号、岁、点…

HSK1 词类 / 量词 细目 : 时量词

Idée générale

Les classificateurs de temps mesurent des unités temporelles : 号/日 (jour du mois), 岁 (ans d'âge), 点 (heure), 分 (minute), 年 (année), 天 (jour de durée).

Structure

Nombre + Classificateur de temps
号 pour les dates, 天 pour les durées, 岁 pour l'âge

Comment l'utiliser

  • 号 pour les dates : 三号 (le 3 du mois), 十月一号 (1er octobre).
  • 岁 pour l'âge : 五岁 (5 ans).
  • 天 pour les durées en jours : 三天 (trois jours).

Exemples

今天是十月一号。
Aujourd'hui, c'est le 1er octobre.
她五岁了。
Elle a cinq ans.
现在三点半。
Il est trois heures et demie.
我在北京住了三天。
J'ai vécu trois jours à Pékin.

Notes

  • 日 est plus formel que 号 pour les dates.
  • 天 sert pour la durée (三天 = pendant 3 jours), 号/日 pour la date calendaire.

Pièges fréquents

  • L'âge s'exprime avec 岁, pas 年 : *她五年 → 她五岁.
  • Ne pas confondre 天 (durée) et 号 (date) : 三天 (3 jours) ≠ 三号 (le 3 du mois).
