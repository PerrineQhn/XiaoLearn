名量词 — （1）专用名量词：本、个、家、口…

HSK1 词类 / 量词 细目 : 名量词

Idée générale

Les classificateurs nominaux sont obligatoires entre un nombre (ou un démonstratif) et un nom. 个 est le plus courant. Autres : 本 (livres), 家 (magasins), 口 (bouches/famille), 块 (morceaux/argent), 件 (vêtements/affaires), 只 (animaux), 元 (yuan).

Structure

Nombre / Démonstratif + Classificateur + Nom

Comment l'utiliser

  • 个 est le classificateur par défaut : 一个人, 三个苹果.
  • 本 pour les livres : 一本书 ; 家 pour les magasins/entreprises : 一家商店.
  • 只 pour les petits animaux : 一只猫, 两只鸟.

Exemples

三本书。
Trois livres.
一个人。
Une personne.
两家商店。
Deux magasins.
这件衣服很漂亮。
Ce vêtement est joli.

Notes

  • 个 peut remplacer la plupart des classificateurs en cas de doute, mais c'est moins élégant.
  • Le classificateur est OBLIGATOIRE en chinois : on ne dit jamais Nombre + Nom directement.

Pièges fréquents

  • Ne jamais dire *一书 : il faut 一本书. Le classificateur est obligatoire.
  • Chaque nom a un classificateur approprié : apprendre le classificateur avec le nom.
