单句 — 主谓句2：形容词谓语句

HSK1 句子成分 / 句型 细目 : 单句

Idée générale

La phrase à prédicat adjectival utilise un adjectif comme prédicat, SANS le verbe « être » (是). 很 sert souvent de lien neutre.

Structure

Sujet + (很 / 非常 / 真) + Adjectif

Comment l'utiliser

  • 很 est quasi obligatoire pour un constat neutre ; sans 很, il y a une nuance comparative.

Exemples

今天很热。
Il fait chaud aujourd'hui.
这个菜非常好吃。
Ce plat est vraiment bon.
中文不难。
Le chinois n'est pas difficile.

Notes

  • 很 + adjectif = constat neutre ; adjectif seul = comparaison implicite.

Pièges fréquents

  • *她是漂亮 → 她很漂亮 : pas de 是 entre sujet et adjectif.
