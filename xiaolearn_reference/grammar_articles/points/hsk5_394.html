HSK5 句子成分 / 特殊句型 细目 : 被动句3

Idée générale

Les **formes passives** (被 / 让 / 叫) mettent l'accent sur ce qui est **subi** par le sujet. On les emploie surtout quand l'événement est **marquant** (souvent négatif, mais pas toujours) ou quand on veut mettre en avant la conséquence pour le sujet.

Éléments concernés

主语+被 叫 让+宾语+给+动词+其他成分

Structure(s)

主语+被
让+宾语+给+动词+其他成分

Exemples

我的手机被人偷了。
On m'a volé mon téléphone.

Pièges fréquents

  • Ne pas confondre la fonction (catégorie / structure) avec le vocabulaire : les mots listés suivent un même comportement, mais le sens précis peut varier.
  • Vérifier la place dans la phrase (avant le verbe, après le verbe, dans le groupe nominal, etc.).
  • Quand un schéma implique un complément (résultat, direction, durée…), éviter une phrase “trop courte” qui sonne artificielle.

Notes utiles

  • 被 — Passif : Met l'accent sur ce qui est subi; l'agent peut être omis si évident.
