因果复句 — 既然……、就……

HSK4 句子的类型 / 复句 细目 : 因果复句

Idée générale

Les **phrases complexes de cause-conséquence** (因果复句) relient une cause et un résultat : 因为…所以… / 由于…因此… / 既然…就…

Éléments concernés

既然…… 就……

Exemples

因为时间不够，所以我们改天再见。
Comme on n’a pas assez de temps, on se voit un autre jour.

Pièges fréquents

  • Ne pas confondre la fonction (catégorie / structure) avec le vocabulaire : les mots listés suivent un même comportement, mais le sens précis peut varier.
  • Vérifier la place dans la phrase (avant le verbe, après le verbe, dans le groupe nominal, etc.).
  • Quand un schéma implique un complément (résultat, direction, durée…), éviter une phrase “trop courte” qui sonne artificielle.

Notes utiles

  • 就 — Conséquence / immédiateté : Très polyvalent; souvent avec des structures conditionnelles ou de rapidité (一…就…).
