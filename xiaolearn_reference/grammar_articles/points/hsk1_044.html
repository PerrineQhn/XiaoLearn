单句 — 主谓句3：名词谓语句

HSK1 句子成分 / 句型 细目 : 单句

Idée générale

La phrase à prédicat nominal utilise un nom ou un groupe nominal comme prédicat pour exprimer la date, l'heure, l'âge ou le prix, sans verbe « être ».

Structure

Sujet + Nom / Nombre (pas de 是)

Comment l'utiliser

  • Pour les dates, heures, âges, prix.

Exemples

今天星期五。
Aujourd'hui c'est vendredi.
现在三点。
Il est trois heures.
他今年二十岁。
Il a vingt ans cette année.

Notes

  • C'est une particularité du chinois : le nom sert directement de prédicat.

Pièges fréquents

  • Ajouter 是 n'est pas faux mais souvent inutile et moins naturel.
