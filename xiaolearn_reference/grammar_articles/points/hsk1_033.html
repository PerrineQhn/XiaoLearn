其他结构类型 — （2）介宾短语

HSK1 短语 / 结构类型 细目 : 其他结构类型

Idée générale

Le groupe prépositionnel (介宾短语) est formé d'une préposition suivie de son complément. Il sert de circonstant et se place AVANT le verbe.

Structure

Préposition + Nom / Pronom

Comment l'utiliser

  • 在+lieu, 从+départ, 和+compagnon, 对+personne concernée.
  • Se place toujours AVANT le verbe principal.

Exemples

在学校学习。
Étudier à l'école.
和朋友一起去。
Y aller avec un ami.
从北京来。
Venir de Pékin.

Notes

  • Le groupe prépositionnel se place AVANT le verbe (inverse du français).

Pièges fréquents

  • *我去了在学校 → 我在学校学习 : le groupe prép. précède le verbe.
