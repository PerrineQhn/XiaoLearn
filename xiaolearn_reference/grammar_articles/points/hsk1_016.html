范围副词 — 都1

HSK1 词类 / 副词 细目 : 范围副词

Idée générale

L'adverbe 都 signifie « tous/toutes » et résume un ensemble mentionné AVANT lui dans la phrase. Il se place toujours avant le verbe.

Structure

Sujet (pluriel / ensemble) + 都 + Verbe / Adjectif

Comment l'utiliser

  • L'ensemble résumé par 都 doit être mentionné AVANT 都.
  • 都 se place toujours directement avant le verbe ou l'adjectif.
  • Pour « les deux », mettre l'ensemble en premier : 两个我都喜欢.

Exemples

我们都是学生。
Nous sommes tous étudiants.
他们都会说中文。
Ils savent tous parler chinois.
苹果和香蕉我都喜欢。
J'aime les pommes et les bananes (les deux).
每天都很忙。
Tous les jours c'est très chargé.

Notes

  • 都 se place AVANT le verbe, pas avant le sujet.
  • L'ensemble résumé (两个, 苹果和香蕉, etc.) doit précéder 都.

Pièges fréquents

  • *都人喜欢 → 所有人都喜欢 : 都 est un adverbe, pas un adjectif.
  • *我喜欢都 → 我都喜欢 : 都 est AVANT le verbe, jamais après.
