数量补语1 — 动词+动量补语

HSK1 句子成分 / 补语 细目 : 数量补语1

Idée générale

Le complément de quantité d'action (动量补语) suit le verbe pour indiquer combien de fois l'action est réalisée. Le classificateur le plus courant est 次 (fois).

Structure

Verbe + Nombre + 次 / 遍 / 下

Comment l'utiliser

  • 次 = fois (général) ; 遍 = fois (du début à la fin) ; 下 = fois (bref).
  • Si le verbe a un objet, celui-ci vient après le complément.

Exemples

我去过两次中国。
Je suis allé(e) en Chine deux fois.
请再说一次。
Veuillez le redire une fois.
请你看一下。
Regarde un instant, s'il te plaît.

Notes

  • Si le verbe a un objet, l'objet vient après le complément : 去了两次中国.

Pièges fréquents

  • Ordre : Verbe + Complément + Objet, pas Verbe + Objet + Complément.
