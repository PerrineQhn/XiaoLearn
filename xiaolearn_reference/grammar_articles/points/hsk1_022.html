引出对象 — 和1、对

HSK1 词类 / 介词 细目 : 引出对象

Idée générale

和 (avec) relie deux personnes dans une action commune. 对 (envers, à) introduit la personne concernée par l'action.

Structure

A + 和 + B + Verbe (ensemble)
Sujet + 对 + Personne + Verbe

Comment l'utiliser

  • 和 pour « avec » : 我和他一起去 (Je vais avec lui).
  • 对 pour « envers » : 她对我很好 (Elle est gentille envers moi).

Exemples

我和他一起去。
Je vais avec lui.
她对我很好。
Elle est gentille envers moi.
我和朋友看电影。
Je regarde un film avec un ami.
老师对学生说。
Le professeur dit aux élèves.

Notes

  • 和 comme préposition (avec) se distingue de 和 comme conjonction (et).

Pièges fréquents

  • 和 ne relie pas deux phrases complètes : *我去了和他去了 est incorrect.
