状语 — 副词、形容词作状语 表示时间、处所的词语作状语

HSK1 句子成分 / 状语

Idée générale

Le complément circonstanciel (状语) se place AVANT le verbe. Il modifie l'action et peut être un adverbe, un adjectif, un groupe prépositionnel ou une expression de temps/lieu.

Structure

Sujet + [Temps + Lieu + Manière] + Verbe

Comment l'utiliser

  • Les expressions de temps et lieu se placent avant le verbe.
  • Ordre typique : Sujet + Temps + Lieu + Manière + Verbe.

Exemples

他每天早上跑步。
Il court tous les matins.
我在图书馆学习。
J'étudie à la bibliothèque.
她很高兴。
Elle est contente.

Notes

  • L'ordre typique est : Sujet + Temps + Lieu + Manière + Verbe.

Pièges fréquents

  • *我去明天 → 我明天去 : le temps vient AVANT le verbe.
