其他结构类型 — （3）方位短语

HSK1 短语 / 结构类型 细目 : 其他结构类型

Idée générale

Le groupe de position (方位短语) combine un nom et un mot de position pour indiquer un lieu précis.

Structure

Nom + 上 / 下 / 里 / 外 / 前 / 后 / 边 / 面

Comment l'utiliser

  • Après 在 pour localiser : 在桌子上 (sur la table).
  • Le mot de position vient APRÈS le nom (ordre inverse du français).

Exemples

桌子上。
Sur la table.
学校里。
Dans l'école.
在商店前面。
Devant le magasin.

Notes

  • Le mot de position vient APRÈS le nom de référence.

Pièges fréquents

  • *在上桌子 → 在桌子上 : l'ordre est Nom + Position.
