主语 — 名词、代词或名词性短语作主语

HSK1 句子成分 / 主语

Idée générale

Le sujet est généralement un nom, un pronom ou un groupe nominal. Il se place en tête de phrase, avant le prédicat.

Structure

Sujet (nom / pronom / groupe nominal) + Prédicat

Comment l'utiliser

  • Le sujet indique de qui ou de quoi on parle ; il précède le verbe/adjectif.
  • Le sujet peut être omis quand le contexte est clair.

Exemples

我喜欢中国。
J'aime la Chine.
这本书很好。
Ce livre est bien.
天气很好。
Le temps est beau.

Notes

  • Le sujet peut être omis si le contexte est clair : (我)知道了.

Pièges fréquents

  • Le sujet est TOUJOURS avant le verbe en chinois (pas d'inversion).
