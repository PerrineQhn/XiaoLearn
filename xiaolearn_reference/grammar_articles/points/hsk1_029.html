基本结构类型 — （2）偏正短语

HSK1 短语 / 结构类型 细目 : 基本结构类型

Idée générale

Le groupe déterminatif (偏正短语) est formé d'un modificateur suivi d'un noyau. Le modificateur qualifie ou restreint le sens du noyau.

Structure

Modificateur + (的/地) + Noyau

Comment l'utiliser

  • Nominal : 中国菜 (cuisine chinoise).
  • Adjectival : 漂亮的花 (une jolie fleur), 很大的房子 (une grande maison).
  • Adverbial : 慢慢地走 (marcher lentement).

Exemples

中国菜。
Cuisine chinoise.
很大的房子。
Une grande maison.
我的朋友。
Mon ami.
慢慢地走。
Marcher lentement.

Notes

  • 的 pour les modif. nominaux/adjectivaux, 地 pour les modif. adverbiaux.

Pièges fréquents

  • Le modificateur vient TOUJOURS avant le noyau en chinois.
