完成态 — 用动态助词“了1”表示

HSK1 动作的态 / 细目 : 完成态

Idée générale

L'aspect accompli s'exprime avec 了 placé après le verbe. Il indique qu'une action est achevée.

Structure

Verbe + 了 + (Nombre + Classif + Objet)

Comment l'utiliser

  • Après le verbe pour signaler l'action accomplie ; l'objet est souvent quantifié.

Exemples

我买了两本书。
J'ai acheté deux livres.
他吃了一个苹果。
Il a mangé une pomme.
她去了上海。
Elle est allée à Shanghai.

Notes

  • 了 marque l'ACCOMPLISSEMENT, pas le passé en soi.

Pièges fréquents

  • Sans quantification, la phrase semble incomplète : 我买了书 → 我买了一本书.
