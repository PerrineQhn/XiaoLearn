定语 — 名词性词语、形容词性短语、数量短语作定语

HSK1 句子成分 / 定语

Idée générale

L'épithète (定语) modifie un nom et se place AVANT le nom, souvent avec la particule 的. Les modificateurs peuvent être des noms, des adjectifs ou des groupes numéral-classificateur.

Structure

Modificateur + (的) + Nom

Comment l'utiliser

  • Adjectif + 的 + nom : 漂亮的花.
  • Nombre + classif + nom : 三本书.
  • 的 peut être omis pour les adjectifs monosyllabiques : 大房子, 好人.

Exemples

漂亮的女孩。
Une jolie fille.
三个学生。
Trois étudiants.
新的衣服。
Un nouveau vêtement.
中国菜。
Cuisine chinoise.

Notes

  • 的 peut être omis pour les adjectifs monosyllabiques courants : 大房子, 好人.

Pièges fréquents

  • Ne pas oublier 的 pour les adjectifs longs : *很漂亮女孩 → 很漂亮的女孩.
